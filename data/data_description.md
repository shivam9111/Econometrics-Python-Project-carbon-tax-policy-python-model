# Data Description

This project uses survey data from Hope et al. (2026) on carbon tax support in the United Kingdom.

The dataset contains demographic variables, treatment assignment, and attitudinal measures related to perceived fairness and unequal treatment.

The original dataset is restricted and cannot be publicly shared.

To reproduce the analysis, users must supply the original Excel file in this directory with the expected sheet names:
- "Data"
- "Data dictionary"
